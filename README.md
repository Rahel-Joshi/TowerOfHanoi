
# Tower of Hanoi

To run the program:

$ git pull https://github.com/Rahel-Joshi/TowerOfHanoi \
$ python main.py

User commands:\
    r: reset configuration\
    p: print configuration\
    q: quit program\
    a: autosolve\
    n1 n2: move disk from n1 to n2
